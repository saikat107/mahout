# Dense Passage Retrieval

### Workflow

#### Step1. Training the Retriever

```
python script_train.py
```

#### Step2. Encode Sequences and Create Index

```
python script_encode.py
```

#### Step3. Retrieve and Evaluate

```
python script_eval.py
```
